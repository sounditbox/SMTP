SMTP-client
Автор: Руслан Мутыгуллин
Требования: Python 3.х, PyQt5
Состав:
    Файл запуска: main.py
    GUI: gui/
    Тестовый документ для проверки фичи с html: html_example.txt
    Класс, отвечающий за построение письма: mail.py
    Класс, отвечающий за работу smtp: smtp.py
Подробности реализации:
    Реализованы следующие фичи:
        Указание произвольных smtp-серверов и их портов
        Указание нескольких адресатов
        Возможность отправить копии и скрытые копии(CC,BCC)
        Передача сообщения в виде text или html
        Прикрепление нескольких вложений произвольного типа
    Если получателей сообщения/копии/скрытой копии несколько
    необходимо вписать их email-адреса через запятую
